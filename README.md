yt_antd_pro是基于Ant Design Pro进行二次封装的前端框架</br>
</br>
主要是适用于传统企业开发的增删改查页面，您可以配合yt_mybaits框架一键运行示例。</br>
</br>
1.安装nodejs</br>
2.进入工程目录 npm install</br>
3.npm start</br>
4.# visit http://localhost:8000</br>
